# Icons for Firefox Extension

You need to create 4 icon sizes for the extension. Place them in this folder with these exact names:

## Required Files

- `icon-16.png` (16x16 pixels) - Toolbar icon, small
- `icon-48.png` (48x48 pixels) - Extension management
- `icon-96.png` (96x96 pixels) - Firefox Add-ons page
- `icon-128.png` (128x128 pixels) - High-DPI displays

## Quick Icon Generation Options

### Option 1: Use Online Generator (Fastest)
1. Go to https://www.favicon-generator.org/

3. Download all sizes and rename appropriately

### Option 2: Use Canva (Free)
1. Go to https://www.canva.com/
2. Create a 128x128 design with magnifying glass or fact-check theme
3. Download as PNG
4. Use https://www.iloveimg.com/resize-image to create other sizes

### Option 3: Use AI Image Generator
1. Use DALL-E, Midjourney, or similar
2. Prompt: "A modern app icon for a fact-checking extension, magnifying glass with checkmark, purple gradient, minimal flat design, 128x128"
3. Resize for other dimensions

### Option 4: Free Icon Resources
- https://icons8.com - Search "fact check" or "verify"
- https://www.flaticon.com - Search "magnifying glass"
- https://heroicons.com - Get SVG and convert to PNG
